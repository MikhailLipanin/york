# [WIP] York

**York** is an open-source generator of Golang projects from scratch. It allows to set your custom settings for further use of generating projects.

## Usage


## Example

```json
{
  "projectName": [
    {
      "cmd": {
        "app": [
          "main.go",
          "doc.txt"
        ]
      }
    },
    ".gitignore",
    "Makefile",
    "Dockerfile",
    "go.mod",
    "go.sum"
  ]
}
```
